# 环境要求

```
node： 20.11.1 或以上版本
npm： 10.2.4 或以上版本
```

# 运行

```
1、全局安装Taro，使用的是Taro V3.6.34，命令如下

npm install -g @tarojs/cli

2、安装依赖库
yarn install

3、启动项目
微信：yarn dev:weapp
钉钉：yarn dev:dd
H5运行：一定要 localhost:8888
```
